Trapping Rain Water

Given n non-negative integers representing an elevation map where the width of each bar is 1, compute how much water it can trap after raining.

Example 1:#

Input: height = [0,1,0,2,1,0,1,3,2,1,2,1]

Output: 6

Explanation: The above elevation map (black section) is represented by the array [0,1,0,2,1,0,1,3,2,1,2,1]. In this case, 6 units of rainwater (blue section) are being trapped.

Solution in Python#

class Solution:
    def trap(self, height: List[int]) -> int:
        maxright = 0 
        maxleft = 0 
        capacity = 0 

        l, r = 0, len(height) - 1
        while l <= r: 
            if maxleft < maxright:
                amount = maxleft - height[l]
                if amount > 0: 
                    capacity += amount
                maxleft = max(height[l], maxleft)
                l += 1
            else: 
                amount = maxright - height[r]
                if amount > 0: 
                    capacity += amount 
                maxright = max(maxright, height[r])
                r -= 1
            
        return capacity

Explanation of Solution

This logic takes advantage of the fact that water can only be trapped between bars if there is a taller bar on both sides. Thus, the algorithm only needs to keep track of the tallest bars on both ends and move inwards. By doing so, it efficiently calculates the total volume of trapped rainwater without the need for complex data structures or algorithms.
